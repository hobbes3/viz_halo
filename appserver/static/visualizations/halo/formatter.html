All numbers in pixels unless otherwise noted.
The canvas width of the halo viz. Increase this and/or panel width/height if you see clipping.
Defaults to width = this.$el.width() * 0.8 (ie 80% of the panel width).
The canvas height of the halo viz. Increase this and/or panel width/height if you see clipping.
Defaults to height = width * 0.8.
The outer radius of the halo viz.
Defaults to radius = width / 2 * 0.55.
The radius to the labels from the center of the halo viz.
Defaults to radius_label = radius * 1.1 (ie 10% further from the outer ring radius).
The thickness of the outer and inner rings.
Defaults to thickness = radius * 0.07.
The maximum allowed radius for circle packing of the inner rings (see d3.pack()).
Defaults to radius_pack = 0.8 * (radius - thickness) (ie 80% of the inner radius of the outer ring).
The spacing/padding between inner rings.
Defaults to padding_pack = radius * 0.1.
The curvature of the ribbon from the inner rings. Setting to 0 disable curvature (meaning the ribbons will be straight lines from the inner rings to the outer ring).
Defaults to ribbon_radius_cp_offset = radius * 0.2.
The d3 category scale for the color of the outer rings. More info (and actual colors) found at d3 category scales and d3 chromatic category scales. Defaults to outer_colors = "schemeCategory20b".
All numbers must range from 0 to 1.
The opacity of the ribbons (while not mouseover).
Defaults to opacity_ribbon = 0.6.
The opacity of all unrelated objects during a mouseover.
Defaults to opacity_fade = 0.1.
All numbers in pixels unless otherwise noted.
The font size of the labels.
Defaults to label_font_size = radius * 0.03.
The spacing between the labels and the lines from the outer rings.
Defaults to label_spacing = radius * 0.01.
The maximum width of the labels before line wrapping is forced.
Defaults to label_wrap_length = radius * 0.7.
Show or hide the inner labels. Defaults to inner_labels = "on".
The scale (from 0 to 1) on how much "space" the inner labels should take up inside.
Defaults to inner_label_scale = 0.9.
All numbers in pixels unless otherwise noted.
The total time (in millisecond) the transition will take during an animation (ie changing ribbon types).
Defaults to transition_duration = 750.
The increment (how far up/down) the labels move during a cycle of the label de-collison check.
Defaults to label_relax_delta = 0.5.
How much to wait (in milliseconds) before executing another cycle of the label de-collison check.
Defaults to label_relax_sleep = 10.
